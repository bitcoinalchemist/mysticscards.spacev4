# mysticscards.space

Divination and esoteric tools on a dark, starlit deck — three hand-built
pages, no build step, no framework.

**→ [mysticscards.space](https://mysticscards.space)**

- **Mystics Cards** — a cardology finder: enter a birthday to find its cards,
  read the Life Script and yearly cycles, and explore the Quadrations spread,
  the four suits, and the planets.
- **I Ching** — cast a six-line reading with playing cards, with the full
  64-hexagram reference and the eight trigrams.
- **Seed Oracle** — a gated teaching journey through BIP39 and the I Ching.

## Built with

Plain HTML, CSS, and JavaScript, served straight from the repository root —
no bundler, no runtime dependencies, no build. Astronomy is computed in the
browser with [Astronomy Engine](https://github.com/cosinekitty/astronomy).
